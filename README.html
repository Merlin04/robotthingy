robotthingy

This is a Python robot for Linux. You need to have these packages installed: inxi python3 espeak You will also need the apt package manager for the update capabilities.

How to run

Run this with: python3 robotthingy.py

Command keywords

Whenever this script sees one of these keywords, it will do something. Here are the keywords:\n time: prints the time on the screen.\n update: runs sudo apt-get update./n terminal: gives you a command prompt. Note: LXterminal is not working. Just use option 1.\n easter egg: ?????????????\n meow: ?????????????\n weather: prints the weather on the screen.\n quit: quits robotthingy.\n shutdown: runs sudo shutdown -h now.\n
